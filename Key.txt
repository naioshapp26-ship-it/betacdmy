## Overview

This project is deployed in **two environments**:

| Environment | Platform | Purpose | Auto-Deploy |
|-------------|----------|---------|-------------|
| **Testing** | Railway | Test new features before going live | ✅ Yes (on git push) |
| **Production** | cPanel | Live customer-facing website | ❌ No (manual deploy script) |

---

## Dual Deployment Architecture

```
Your Local Machine
       ↓
   git push origin main
       ↓
   GitHub Repository
       ├─────→ Railway (Auto-deploys automatically)
       │       Test your changes here first
       │
       └─────→ cPanel (Manual deploy using script)
               Production site
```

**Workflow:**
1. Make code changes locally
2. Push to GitHub: `git push origin main`
3. Railway automatically deploys (wait 2-3 minutes)
4. Test on Railway to ensure everything works
5. If tests pass, run the deploy script for cPanel
6. Production is updated!

## How to Deploy Changes

### Option A: Deploy from Your Local Computer (Recommended)

**Single Command Deployment:**

```bash
sshpass -p 'Beta_123@#' ssh -o StrictHostKeyChecking=no edunaiosh@vmi2553862.contaboserver.net 'bash /home/edunaiosh/public_html/deploy.sh'
```

**What this does:**
1. Connects to cPanel server via SSH
2. Runs the deploy script
3. Pulls latest code from GitHub
4. Installs dependencies
5. Restarts the application

**Time:** 30-60 seconds

---

### Option B: Deploy from cPanel Terminal

**Steps:**
1. Go to: https://cpanel.edunaiosh.com/
2. Login with: `edunaiosh` / `Beta_123@#`
3. Search for "Terminal" and open it
4. Run:
```bash
bash /home/edunaiosh/public_html/deploy.sh
```

---

### What the Deploy Script Does

The deploy script (`/home/edunaiosh/public_html/deploy.sh`) automatically:

```bash
#!/bin/bash

# 1. Navigate to app directory
cd /home/edunaiosh/public_html/betacdmy-app

# 2. Pull latest code from GitHub
git pull origin main

# 3. Install new dependencies (if any)
npm install --production

# 4. Restart the Node.js application
pkill -f "node server.js"
nohup node server.js > /home/edunaiosh/app.log 2>&1 &

# 5. Show success message
echo "✅ Deployment complete!"
echo "🌐 Site: https://www.betacdmy.com"
```

## Troubleshooting

### Issue: Deploy Script Shows "Git pull failed"

**Cause:** GitHub SSH authentication issue

**Solution:**
```bash
# Test SSH connection to GitHub
sshpass -p 'Beta_123@#' ssh -o StrictHostKeyChecking=no edunaiosh@vmi2553862.contaboserver.net "ssh -T git@github.com"

# Should see: "Hi KarimWajihKMW! You've successfully authenticated..."
```

If authentication fails, check if the deploy key is still active at:
https://github.com/KarimWajihKMW/LMS-temp-new/settings/keys

---

### Issue: Website Shows Blank Page

**Cause:** Application crashed or not running

**Solution:**
```bash
# Check if app is running
sshpass -p 'Beta_123@#' ssh -o StrictHostKeyChecking=no edunaiosh@vmi2553862.contaboserver.net "ps aux | grep 'node server.js' | grep -v grep"

# Check application logs for errors
sshpass -p 'Beta_123@#' ssh -o StrictHostKeyChecking=no edunaiosh@vmi2553862.contaboserver.net "tail -50 /home/edunaiosh/app.log"

# Restart application
sshpass -p 'Beta_123@#' ssh -o StrictHostKeyChecking=no edunaiosh@vmi2553862.contaboserver.net 'bash /home/edunaiosh/public_html/deploy.sh'
```

---

### Issue: npm install fails

**Cause:** Dependency conflicts or npm registry issues

**Solution:**
```bash
# SSH into server
sshpass -p 'Beta_123@#' ssh -o StrictHostKeyChecking=no edunaiosh@vmi2553862.contaboserver.net

# Navigate to app directory
cd /home/edunaiosh/public_html/betacdmy-app

# Clear npm cache and reinstall
rm -rf node_modules package-lock.json
npm install --production

# Restart app
pkill -f "node server.js"
nohup node server.js > /home/edunaiosh/app.log 2>&1 &
```

---

### Issue: Database Connection Errors

**Cause:** Database credentials changed or PostgreSQL not running

**Solution:**
```bash
# Test database connection
sshpass -p 'Beta_123@#' ssh -o StrictHostKeyChecking=no edunaiosh@vmi2553862.contaboserver.net "PGPASSWORD='Betacdmy_12345' psql -h localhost -U betacdmy -d edunaiosh_betacdmy -c 'SELECT version();'"

# Check .env file
sshpass -p 'Beta_123@#' ssh -o StrictHostKeyChecking=no edunaiosh@vmi2553862.contaboserver.net "cat /home/edunaiosh/public_html/betacdmy-app/.env | grep DATABASE_URL"
```

---

### Issue: Changes Not Showing on Website

**Possible Causes:**
1. Forgot to run deploy script after pushing to GitHub
2. Browser cache showing old version
3. CDN caching (if using CDN)

**Solution:**
```bash
# 1. Verify latest commit on server
sshpass -p 'Beta_123@#' ssh -o StrictHostKeyChecking=no edunaiosh@vmi2553862.contaboserver.net "cd /home/edunaiosh/public_html/betacdmy-app && git log --oneline -1"

# Compare with GitHub latest commit
git log --oneline -1

# 2. If different, run deploy script
sshpass -p 'Beta_123@#' ssh -o StrictHostKeyChecking=no edunaiosh@vmi2553862.contaboserver.net 'bash /home/edunaiosh/public_html/deploy.sh'

# 3. Clear browser cache or test in incognito mode
```
